**Zed Compiler Version 0.0.1**

The Zed Compiler Is Back!, Currently Only The Parser And Lexer Are Implemented  
Currently Zed Is Only Avalible For Windows, "So How Do I Set It Up?"  
To Setup Zed Either Clone The Repo Or Dowload The Code As A Zip  
Run In Your Terminal:  
`C:Path_To_The_Zed_Folder> .\Setup.exe`  
To Test Everything Is Good To Go Run:  
`C:Path_To_The_Zed_Folder> python Zedc.py --version` 

**!!Note Zedc Is Currently Not PreCompiled You Can Compile It With PyInstaller!!, Happy Coding**
